Changelog

New updates and improvements.

v2.4.0 Feature March 25, 2026

Added two-factor authentication with TOTP support.

  • QR code setup flow for authenticator apps
  • Backup recovery codes generated on enable
  • Option to require 2FA for all team members
v2.3.2 Fix March 18, 2026

Fixed session expiration not redirecting to login page. Users on expired sessions would see a blank page instead of the login form.

v2.3.1 Improvement March 10, 2026

Improved password strength indicator with real-time feedback and entropy calculation. Now shows estimated crack time.

v2.3.0 Feature February 28, 2026

Introduced magic link authentication as an alternative to passwords.

  • One-click email login with secure token
  • Configurable link expiration (default 15 min)
  • Rate limiting to prevent abuse
v2.2.1 Fix February 15, 2026

Fixed OAuth callback failing when redirect URI contained query parameters. Added proper URL encoding for state parameter.

v2.2.0 Improvement February 1, 2026

Redesigned all form templates with improved mobile responsiveness and accessibility.

  • WCAG 2.1 AA compliant focus indicators
  • Better touch targets on mobile (min 44px)
  • Reduced motion support via prefers-reduced-motion
